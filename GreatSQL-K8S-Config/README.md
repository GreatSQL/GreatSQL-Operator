# Kubernetes 配置文件

鉴于Kubernetes系统的管理主要是通过声明式的yaml配置文件来实现的,而yaml作为一种数据序列化格式,对块式内容的语法格式有较为严格的缩进要求。一次简单的缩进错误就可能导致配置失败。

为减少此类低级错误影响,专门建立存放平台相关的所有Kubernetes配置文件。
